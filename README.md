Yonder!
=======

Scupr repo is the server side of [Yonder!] (https://github.com/Berta-G/yonder-client).

Built together with Andrey Krahmal ([@plawtr])in 10 days as the final project at [Makers Academy].

Technologies:
-------------
- Built in Rails framework.
- Geokit
- Paperclip
- Amazon's S3
- Postgresql
